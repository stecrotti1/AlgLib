size_t upo_io_read_line ( FILE *  stream,
char **  lineptr,
size_t *  size 
)

Reads an entire line, storing it in string pointed by lineptr.

Parameters
streamThe stream from which the line has to be read.
lineptrPointer to the string where the line will be stored. If it is NULL, the function will allocate a buffer for containing the line, which must be freed by the client; otherwise, this parameter must point to a malloc-allocated string that will be eventually enlarged if its size does not fit the line.
[output]
sizePointer to the size of the line; it will be updated by the function in the case lineptr is NULL or more memory has been allocated for storing the line.
[output]
Returns
The number of read characters on success, or -1 on error.

The resulting buffer *lineptr is null-terminated and includes the newline char, if a newline delimiter was found.

Precondition
stream != NULL
lineptr != NULL
size != NULL
